# Call-Manager-Bot
QQ频道召唤管理机器人
### 菜鸟练手之作，屎山代码。
### SDK：[botpy](https://github.com/tencent-connect/botpy) 
### 参考（ctrl-c + ctrl-v）[bot-onduty](https://github.com/tencent-connect/bot-onduty) 


### 环境配置
需使用`pip`安装以下模块：
``` 
qq-bot 0.85            # QQ频道官方python SDK
tortoise-orm 0.19.0    # SQLite3 ORM
schedule 1.1.0         # 定时执行任务
```
